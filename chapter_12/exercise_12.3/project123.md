# Проект класса Stock123  
Закрытые члены:  
  - название компании;  
  - количество акций;  
  - цена акции;  
  - стоитмость пакета.  

Конструкторы/деструктор:    
  - конструктор по умолчанию;  
  - конструктор с аргументами;  
  - деструктор.

Методы:  
  - покупка пакета акций;  
  - продажа пакета акций;  
  - изменение цены акции;  
  - определение топового пакета.  
Друзья:  
  - оператор вывода.
